# Magical 入力補助

ぴあのチケット申し込みフォームへの個人情報入力を自動化するスクリプトです。

> 現状は2名申し込みまでしか考慮していません

---

## 方法1: フォーム入力スクリプト（Node.js / PowerShell）

フォームを埋める JS を生成し、ブラウザのコンソールで実行します。

### 必要なもの

**Node.js** または **PowerShell 7 以降**

- Node.js は概ね最新のもので動作確認済み
- PowerShell は 7.4 で動作確認済み
- **「Windows PowerShell」（バージョン 5.1）は使えません。** PowerShell 7 以降を別途インストールしてください。

### 設定ファイルの準備

`settings.default.json` を `settings.json` にリネームし、自分の情報を入力してください。

```
settings.default.json  ->  settings.json
```

> 住所分割については、郵便番号入力時に自動で入力されるフォームの仕様に合わせて調整してください。

### 実行

**Node.js の場合:**

```
npm install
node app.ts
```

**PowerShell 7 の場合:**

```
./app.ps1 -ExecutionPolicy Bypass
```

### 使い方

出力された JS を、個人情報入力画面でブラウザの開発者ツール（`F12`）のコンソールに貼り付けて実行します。

フォームが自動入力されるので、内容を確認してから送信ボタンを押してください。

## 方法2: Playwright による自動化

画面遷移もある程度自動で行います。要所要所で手動操作が必要です。

> U-18は未対応です

### 必要なもの

Node.js に加えて Playwright が必要です。

```
npm install
npx playwright install
```

### 設定ファイルの準備

`settings.json` に以下の項目を追加してください。

| キー | 説明 |
|------|------|
| `url` | 申し込みページの URL |
| `セブン-イレブンで支払い` | コンビニ払いなら `true`、クレカ払いなら `false` |
| `credit` | クレジットカード情報（任意） |
| `pia` | ぴあの ID とパスワード |

```jsonc
{
  // 申し込みの URL
  "url": "https://pia.jp/piajp/v/.../",

  // 支払い方法（コンビニ決済なら true）
  "セブン-イレブンで支払い": true,

  // クレジットカード情報（任意）
  "credit": {
    "number": "1234567890129921",
    "name": "MIKU HATSUNE",
    "year": "2026",
    "month": "01",
    "cvv": "123"
  },

  // ぴあ ID とパスワード
  "pia": {
    "id": "test@example.com",
    "password": "1234"
  }
}
```

### 実行

```
node main.mjs
```

### 手動で行う箇所

以下の操作は自動化されていないため、手動で行ってください。画面遷移後に処理を再開します。

- 個人情報入力後の送信ボタンクリック
- 公演の選択
- 座席の選択（第一希望の枚数のみアシストあり）
- クレジットカード決済の場合、カード情報入力後のボタンクリック
- ぴあのログイン実行ボタンクリック
- 確認画面の画像数字入力と完了ボタンクリック
